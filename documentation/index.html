Welcome to 4K First Legal Aid

The product aims to provide a solution to different types of real-life legal scenarios which an individual is facing. It will assist them by showing the legitimate solution regarding their legal queries. An individual needs to enter his complaint and the product will assist whether any previous cases have been reported, which have similar context and inform him about different actions he should take. Different datasets stored in Hadoop cluster and analyzed on MapReduce cluster will be used to train the machine learning model. This model will be used to analyze and predict the accurate solutions based on their legal queries.

Requirements

  • RHEL 7+
  • Python 3.6+
  • Docker
  • Ansible
  • Hadoop

Packages

  • face_recognition - pip3 install face_recognition
  • OpenCV 3+ - pip3 install opencv-python
  • dlib
  • web sockets - pip3 install websockets

Note: The project will not work until following is done

  • Disable SELINUX: setenforce 0

  • Disable Firewall: iptables -F

  • Run face authentication python program: ./cgi-bin/socket_imgauth.py &

  • Run speech processing python program: ./cgi-bin/socket_voice.py &

  • ServerIP: 192.168.43.98 (Not used in dynamic way, so need to change manually in some files)
