Matplotlib

For plotting graphs on Linux machines, various Python dependencies need to be installed first. To plot graph in real-time in a website, a graph is plotted first, then, saved to a JPEG image file and that file is shown in the webpage.

Now, whenever client enters a query, the script matches the words with our pre-defined dictionary and proceed accordingly.

Requirements

Packages

  • cgi
  • pandas - pip3 install pandas
  • numpy - pip3 install numpy
  • matplotlib - pip3 install matplotlib
  • tkinter - yum install tkinter

Working

Step 1: Query is matched with our dictionary.

Step 2: A function is called for plotting graphs that plots and save the graphs in a directory, which is then loaded on the webpage.

Problems

Problem 1: Load different datasets for different query.

Solution: We cleaned, filtered and assigned various datasets to different variables and we put these variable names in another dataset table. However, at one point in our code, we needed to get the contents of the variable given its name.

globals() is used to call global variable defined earlier. It changes str() to that datatype which is already defined.

Problem 2: Plot graph on linux.

Solution: To solve this problem, we used tkinter but it requires various variables and environment set-ups. So we added a line: matplotlib.use('Agg') before we imported tkinter and it worked like a charm.
