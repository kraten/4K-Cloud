Highlights

  • Start, stop and launch containers of different OS
  • Monitor container stats like RAM, Network Usage, etc. in real-time

Problems

Problem 1: How to get docker container stats in real-time?
Solution: Any docker container stats can be viewed using the command
# docker stats container_name

There is a problem with the above command as it do show live stats but it continues to run indefinitely. However, there is an option to show the live stats only of that instant.
# docker stats container_name --no-stream

Problem 2: How to update container stats displayed on the HTML page every second?
Solution: Use HTML header meta tag to reload the webpage after every fixed duration of seconds
<meta http-equiv = "refresh" content ="1"/>
